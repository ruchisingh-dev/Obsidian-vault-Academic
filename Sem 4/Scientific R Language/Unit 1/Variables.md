## Variables in R
- **Variables** is used to store the data with named locations that your programs can manipulate.
- A variable name can be a combination of **letters**, **digits**, **period** and **underscore**.

| **Valid variable** | **Invalid variable** |
| ---- | ---- |
| total | tot@l |
| sum | Sum |
| .count | _count |
| .count.total | TRUE |
| Var | .0ar |

**Examples:**
```r
x<-10
.y<-20
z<-x+.y
print(z)
# Output: 30
```
